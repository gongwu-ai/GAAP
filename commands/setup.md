# GAAP Setup

Run the interactive setup wizard:

```bash
python3 "${CLAUDE_PLUGIN_ROOT}/scripts/setup.py"
```

This will guide you through:
1. Configuring Feishu webhook URL
2. (Optional) Setting up message compression with LLM

After setup, restart Claude Code to apply the changes.
